# Dark Mode Toggle

A theme switching example demonstrating:
- Local component imports
- npm packages (lucide-react, motion)
- CSS custom properties for theming
- State persistence patterns

## Files

- `spec.md` - The interactive PRD document
- `ThemeToggle.jsx` - Animated toggle switch
- `ThemeProvider.jsx` - Theme context provider
- `styles.css` - CSS custom properties for themes

## Usage

Open `spec.md` in VS Code and press `Cmd+K P` for Interactive Preview.
