# Notification System

A complete notification system example demonstrating:
- Local component imports
- npm packages (lucide-react, motion)
- Custom CSS
- Interactive state management

## Files

- `spec.md` - The interactive PRD document
- `NotificationBell.jsx` - Bell icon with badge
- `NotificationItem.jsx` - Individual notification row
- `styles.css` - Custom animations and styles

## Usage

Open `spec.md` in VS Code and press `Cmd+K P` for Interactive Preview.
